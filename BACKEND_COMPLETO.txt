**Modelos Implementados:**
- ✓ Ideal (Lei de Raoult)
- ✓ NRTL (Non-Random Two-Liquid)

- ✓ Diagramas T-xy (binários)

**API Endpoints:**
- POST /elv/bubble-point-pro
- POST /elv/dew-point-pro
- POST /elv/flash-pro
- POST /elv/pxy-diagram-pro
- POST /elv/txy-diagram-pro

**Status:** ✅ COMPLETO E TESTADO

## 🚀 COMO USAR
================================================================================

### Iniciar servidor:
python run.py

### Acessar módulos:
- Homepage: http://localhost:5000
- ELV: http://localhost:5000/elv
- ELL: http://localhost:5000/ell
- ESL: http://localhost:5000/esl

### Exemplo de requisição API:

POST http://localhost:5000/elv/bubble-point-pro
Content-Type: application/json

{
  "components": ["Water", "Ethanol"],
  "x": [0.5, 0.5],
  "temperature": 353.15,
  "model": "PR"
}
